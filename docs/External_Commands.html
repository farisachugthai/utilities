magic_aid¶

Create a script for searching through aliases.

IPython Aliases¶

By invoking %rehashx on IPython’s startup, there are regularly more than 1000 aliases in the namespace.

This makes basic lookups slightly more involved.

pyutil.magic_aid.phelp(obj, shell=None)[source]¶

Pipe obj through pydoc to pynvim with ft set to man.

Parameters

  • obj (object) – Object to view documentation for in pynvim with pydoc.

  • shell (IPython.core.interactiveshell.InteractiveShell) – Global IPython instance.

Examples

>>> from pyutil.magic_aid import phelp
>>> from IPython import get_ipython
>>> import matplotlib as mpl
>>> _ip = get_ipython()
>>> object_of_interest = mpl
>>> phelp(object_of_interest, _ip)
